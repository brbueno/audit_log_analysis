IT Audit Log Analysis with Python

Overview
This project demonstrates how Python can be used to automate IT audit procedures by analyzing access logs and identifying potential risk indicators.

Features
- Detection of access outside business hours
- Identification of weekend access
- Highlighting high-privilege users
- Automated generation of audit findings

Technologies
- Python
- Pandas

Use Case
This project simulates a real-world IT audit scenario where access logs are analyzed to identify anomalies and potential security risks.

Output
A CSV file containing flagged records for further audit investigation.
